Découvrez comment utiliser Firepoz pour gérer vos ventes, achats, versements et créations de produits.

Gérer les Ventes

Pour enregistrer une vente :

  • Accédez à la page Ventes.
  • Scannez le code-barres EAN-13 du produit ou saisissez-le manuellement.
  • Confirmez la quantité et validez la transaction.
  • Consultez l'historique des ventes dans le tableau.
Aller à Ventes

Gérer les Achats

Pour ajouter un achat :

  • Allez à la page Achats.
  • Ajoutez un produit via son code EAN-13 ou en le sélectionnant.
  • Indiquez la quantité et le prix d'achat.
  • Validez pour mettre à jour le stock.
Aller à Achats

Gérer les Versements

Pour enregistrer un versement :

  • Naviguez vers la page Versements.
  • Saisissez les détails du versement (montant, date, bénéficiaire).
  • Confirmez pour enregistrer.
  • Consultez les versements dans l'historique.
Aller à Versements

Créer des Produits

Pour ajouter un nouveau produit :

  • Allez dans la section Gestion des Produits.
  • Cliquez sur Ajouter un produit.
  • Remplissez les champs : nom, code EAN-13, prix, stock initial.
  • Validez pour ajouter le produit au catalogue.
Aller à Gestion

Questions Fréquentes

Comment scanner un code-barres ?

Utilisez un scanner compatible ou la caméra de votre appareil via l'interface de Firepoz. Assurez-vous que le code EAN-13 est bien lisible.

Puis-je modifier une vente après validation ?

Oui, allez dans l'historique des ventes, sélectionnez la transaction et utilisez l'option de modification (sous réserve de permissions).

Comment contacter le support ?

Envoyez un email à support@firepoz.com ou utilisez le formulaire de contact dans l'application.
